# Invitations

## Run

1. Clone the repository:
    ```bash
    git clone https://github.com/wiktoriasw/Invitations.git
    ```

2. Navigate to the project directory:
    ```bash
    cd Invitations
    ```

3. Install the required libraries:
    ```bash
    pip install -r requirements.txt
    ```

4. Run the server with::
    ```bash
    fastapi dev main.py
    ```

## Tests

1. Run tests:
    ```bash
    pytest
    ```
